# Zegar UFC

Klon zegara wykorzystywanego podczas walk w największej federacji MMA. Jednakże projekt umożliwia wprowadzenie modyfikacji dotyczących m.in. nazwisk zawodników, liczby rund, kategorii wagowej itp.
